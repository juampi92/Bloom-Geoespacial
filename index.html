Bloom Filter

El Filtro Bloom es una estructura de datos probabilística, para hacer el espacio más eficiente. Concebida por Burton Howard Bloom en 1970, el Filtro Bloom es utilizado para chequear si un elemento es miembro de un conjunto. Los falsos positivos son posibles, pero no lo son los falsos negativos. Se pueden agregar elementos al conjunto, pero no eliminarlos (salvo que se ataje con un contador de positivos para cada celda). Mientras más elementos haya, mayor es la posibilidad de falsos positivos.

Geospatial Index

Un índice Geoespacial, es un índice capaz de determinar una posición geográfica a partir del mismo. Se crean utilizando una encodificación llamada GeoHash, que devuelve un número de 2 * precisión bits. Mientras mayor sea la precisión, menor es el error que existe entre el índice y las coordenadas que se quieren guardar, pero mayor es el tamaño del índice. Utilizando esta tecnología, se pueden buscar elementos dentro de cierta zona, o cercanos a otro punto de una forma eficiente.
